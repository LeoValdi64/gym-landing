# PowerFit Gym - Landing Page

A professional gym/fitness center landing page built with Next.js 16, React 19, and Tailwind CSS.

## ‚ú® Features

### Page Sections
- **Navigation** - Fixed header with logo, nav links, and mobile hamburger menu
- **Hero** - Bold headline with gradient text, CTAs, and stats (5000+ members, 50+ classes, 15+ trainers)
- **Membership Plans** - 3 tiers (Starter $29, Pro $59, Elite $99) with features and "Most Popular" badge
- **Classes Schedule** - Interactive day selector showing classes for each day of the week
- **Trainers** - 6 expert trainers with photos, bios, and specialty tags
- **Facilities** - 6 world-class amenities (Weight Floor, Cardio Zone, Studios, Recovery, Sauna, Nutrition Bar)
- **Testimonials** - 3 member success stories with star ratings
- **Location/Hours** - Address, operating hours, phone, email with icons
- **Contact Form** - Name, email, phone, message fields with success feedback
- **Footer** - Quick links, support, social media icons

### Design
- Dark theme (#0a0a0a background)
- Red/orange gradient accents (#ef4444 to #f97316)
- Hover animations on cards
- Bold typography with gradient text effects

### SEO
- Full Schema.org LocalBusiness structured data
- OpenGraph and Twitter Card meta tags
- Semantic HTML with proper heading hierarchy
- Keywords, descriptions, and canonical URL

### Responsive
- Mobile-first approach with Tailwind breakpoints
- Collapsible mobile navigation
- Grid layouts that adapt from 1 to 3 columns
